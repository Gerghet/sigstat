ParseRules(String)

Parses the production rules from the given string. The first nonterminal symbol in the left hand side of the first rule is treated as the start symbol. Nonterminal symbols must be enclosed into square brackets, eg [Symbol]. All other characters are treated as terminal symbols. [Verifier] -> [Feature]_[Classifier] [Feature] -> X,Y,P,XY,YXP,XP,YP [Classifier] -> DTW, HMM
